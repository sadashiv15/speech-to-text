# Speech to Text — Pelocal AI Tech Hiring Assignment

Real-time multilingual speech transcription web app using OpenAI Whisper.

## Tech Stack
- Python 3.14
- FastAPI + WebSockets
- OpenAI Whisper (self-deployed, open-source)
- ffmpeg for audio conversion

## Features
- Real-time transcription via WebSocket streaming
- Supports Hindi, Marathi, English and Hinglish
- Live waveform visualizer
- Silence detection to skip empty audio
- Language selector (switch mid-session)

### Prerequisites
